# adhesive-lab

粘合剂配方计算与管理系统

## 项目简介
粘合剂配方管理、成分数据库、配比自动计算、实验性能录入与分析。

## 核心功能
1. 配方管理
2. 成分数据库
3. 配比自动计算
4. 性能数据管理
5. 数据导出与分析
6. 多用户协作 & 权限

## 主要技术栈
- 前端：React/Vue
- 后端：Python/Node.js
- 数据库：PostgreSQL/MySQL

## 目录结构示例
- README.md        项目说明
- schema.sql       数据库结构
- ratio_calculator.py 配比计算脚本
- FormulaForm.tsx  前端管理页面示例

## 快速开始
1. 克隆仓库
2. 安装依赖并运行后端
3. 运行前端
